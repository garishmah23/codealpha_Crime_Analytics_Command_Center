# 🕵️ Crime Analytics Command Center

A powerful Crime Analytics Dashboard built using **Python, Tkinter, Pandas, and Matplotlib**. This application allows users to load crime datasets, visualize crime trends, analyze crime categories, and explore data through an interactive graphical interface.

---

## 📌 Project Overview

Crime Analytics Command Center is a desktop-based data visualization application designed to transform raw crime data into meaningful insights.

The dashboard helps users:

* Analyze crime trends over time
* Identify the most common crime categories
* Explore datasets through an interactive table
* Generate visual reports using charts and graphs

This project demonstrates skills in:

* Data Analytics
* Data Visualization
* GUI Development
* Python Programming
* Business Intelligence

---

## 🚀 Features

### 📂 Load Crime Dataset

* Import CSV files containing crime records.
* Automatically display data in a table.

### 📈 Crime Trend Analysis

* Visualize yearly crime trends.
* Identify increases or decreases in crime rates.

### 📊 Crime Type Distribution

* Analyze the frequency of different crime categories.
* Display results using bar charts.

### 🖥 Interactive Dashboard

* User-friendly graphical interface built with Tkinter.
* Dynamic chart generation.

## 📋 Dataset Format

The CSV dataset should contain the following columns:

```csv
Year,Crime_Type,Count
2020,Theft,120
2020,Robbery,45
2021,Theft,150
2021,Fraud,70
2022,Cyber Crime,95
```

### Column Description

| Column     | Description              |
| ---------- | ------------------------ |
| Year       | Crime occurrence year    |
| Crime_Type | Type of crime            |
| Count      | Number of reported cases |
